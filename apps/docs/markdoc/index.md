## What is Marzahn Dev?

Marzahn Dev is a developer group founded by Leon Marzahn, that's providing smart libraries and utilities
for major frameworks like Angular, React and co. We provide libraries with the latest tech stack and
provide useful functionality like:
- i18n (Internationalization)
- Virtual Scroll (WIP)
- Datatables (WIP)
